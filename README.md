# Tech Stack

- **Vue 3**
- **Vue Router**
- **TailwindCSS**
- **Axios**
- **@formkit/auto-animate**
- **Composition API**

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
